设计说明（给开发同学）

配色逻辑
8 个高饱和糖果色 --candy-*，每个工具用一个纯色作为卡片整体背景（而非小图标点缀），色彩本身就是视觉主角，参照 Super Simple Songs 的角色用色方式。
卡片结构：上色块 + 下白底
.card-top（纯色）+ .card-bottom（白底，向上圆角叠在色块上）模拟"贴纸卡片"，文案放白底上保证可读性，标题放色块上靠 text-shadow 保证可读性。
按钮的"按压"质感
box-shadow: 0 4px 0 0 模拟实体按键厚度，hover/active 时位移+缩小阴影，模仿真实按下的物理反馈，呼应儿童向 UI 常见的"糖果按钮"手法。
背景装饰：云朵+太阳
固定定位的纯 CSS 云朵和太阳形状（.cloud / .sun），营造天空场景，呼应 Super Simple Songs 标志性的简笔云朵元素，全部用纯色圆形拼接，无需图片资源。
字体：Baloo 2 + Quicksand
替换上一版的 Fredoka，改用更圆润饱满的 Baloo 2 做标题（笔画粗、转角圆），Quicksand 做正文，两者都属于"圆体"家族，统一可爱基调。
侧边栏改回浅色
上一版深色侧边栏偏"成熟科技感"，这版改回白底+天蓝背景，每个导航项 hover/active 用对应工具的浅色底+深色字，和卡片配色呼应，整体更轻快。
